# Administrator Operations Guide

This guide assists system administrators, HODs, and finance officers managing campus operations.

---

## 🏛️ Multi-Role Switcher (HOD, Finance, Placement)

Administrators have access to a top header switcher to audit different campus sections:

### 1. HOD Dashboard
*   **Accreditation Checklist**: Verify compliance task states.
*   **Load Balancing**: Audits instructor workloads. Click **"Balance Load Matrix"** to trigger AI suggestions (e.g. reallocating labs).

### 2. Finance Hub
*   View collection rate indicators and pending fee rosters.
*   Use the **AI Financial Assistant** to forecast Q3 collections.

### 3. Placement Suite
*   Audit student eligibility checklists matching target corporate parameters.
*   Check average and highest salary brackets.

---

## 🖥️ Command Palette Binds (`Ctrl+K`)

Press `Ctrl+K` or `Cmd+K` anywhere inside the dashboard:
*   Launches the universal command modal.
*   Select custom routines (e.g. `dropout --cse`) to invoke corresponding AI tools instantly.
